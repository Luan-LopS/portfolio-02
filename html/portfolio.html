Automatização para estração de alertas do Zabbix

Desenvolvi uma automatização de extração de um arquivo.csv que é utilizado na confecção de um excel para realização da Dayle. Este escripte executa as tarefas:

  1. Iniciar uma interface gráfica para realizar o login passando usuário e senha.
  2. Com usuário e senha o executável abre o navegador e realiza o login, após isso coloquei um try para verificar se foi realizado o login caso não pede denovo o usuário e senha.
  3. Login com sucesso é extraido o csv
  4. Após isso é consumido o csv pelo excel, tratado e estilizado conforme o necessário.
