Automatização para extração de alertas do Zabbix

Desenvolvi uma automação que extrai arquivos .csv do Zabbix, utilizados na confecção de relatórios em Excel para a rotina diária da equipe. O script realiza as seguintes tarefas:

  1. Abre uma interface gráfica para o usuário inserir login e senha.
  2. Com os dados, abre o navegador e realiza o login. Caso falhe, solicita as credenciais novamente.
  3. Após login bem-sucedido, realiza o download automático do arquivo .csv.
  4. O arquivo é consumido por uma planilha Excel que trata, organiza e estiliza os dados automaticamente.
